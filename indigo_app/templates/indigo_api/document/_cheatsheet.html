PREFACE

PREAMBLE

{% endblock %} {% block cheat-longtitle %}
Markup the long title

Indicate the long title of the work using LONGTITLE. This usually comes just before the sentence "BE IT ENACTED..."

LONGTITLE To establish ...
To establish ...
{% endblock %} {% block cheat-body %}
Starting the body of the document

Start the body of the document with BODY. This comes after the Preface and Preamble.

BODY

This is the body of the Act.
{% endblock %} {% block cheat-chapter %}
Starting a new Chapter

Start a Chapter with Chapter and a number. The title is optional.

Chapter 2 - Interpretation

This is the body of the Chapter.

Chapter 2
Interpretation

This is the body of the Chapter.
{% endblock %} {% block cheat-part %}
Starting a new Part

Start a Part with Part and a number. The title is optional.

Part 2 - Interpretation

This is the body of the Part.

Part 2 – Interpretation

This is the body of the Part.
{% endblock %} {% block cheat-subpart %}
Starting a new Subpart

A subpart is a grouping container, like a Part, but without "Part" in the generated heading.

Start a subpart with SUBPART and a number. The title is optional.

SUBPART A - Party and costs

This is the body of the Subpart.

A – Party and costs

This is the body of the Subpart.
{% endblock %} {% block cheat-crossheading %}
Include a Crossheading

Use CROSSHEADING to include a heading that isn't a chapter or a part, and when the grouping of child elements is unclear.

CROSSHEADING E. Control under a Scheme
E. Control under a Scheme
{% endblock %} {% block cheat-section %}
Starting a Section

Start a numbered section with the number and an optional title. The title should start with a capital letter and then be all in lowercase.

2. Definitions

This is the body of the section.

2. Definitions

This is the body of the section.
{% endblock %} {% block cheat-subsection %}
Starting a Subsection

Start a numbered subsection with the subsection number in brackets. Subsections do not have titles.

(2) The content of subsection 2.
(2)The content of subsection 2.
{% endblock %} {% block cheat-blocklist %}
Create a numbered list (a blocklist)

Create a numbered list with each item's number in brackets. Indent nested lists with four spaces (or a TAB).

The following items are a blocklist:

(a) first item

(b) second item

    (i) nested item (b)(i)

    (ii) nested item (b)(ii)

(c) third item
The following items are a blocklist: (a) first item (b) second item (i) nested item (b)(i) (ii) nested item (b)(ii) (c) third item
{% endblock %} {% block cheat-special %}
Insert a special character

See guidance on using special characters in the Editor Guide.

{% endblock %} {% block cheat-italics %}
Make text italics

Display a word or a sentence as italics using //word//. Do this for Latin, foreign and other special terms.

For editorial remarks, use [[remark]] instead.

The situation is same, //ceteris paribus//.
The situation is the same, ceteris paribus.
{% endblock %} {% block cheat-bold %}
Make text bold

Display a word or a sentence as bold using **word**. Do not use for headings or defined terms.

The situation called for **bold terms**.
The situation called for bold terms.
{% endblock %} {% block cheat-underline %}
Underline text

Underline a word or a sentence using two underscores on either side: __word__. Do not use for headings.

The situation called for __underlined text__.
The situation called for underlined text.
{% endblock %} {% block cheat-weblink %}
Add a link to a webpage

Link to a webpage using [link text](http://example.com/the-link).

Indigo is created by [Laws.Africa](https://www.laws.africa/).
Indigo is created by Laws.Africa.
{% endblock %} {% block cheat-aknlink %}
Add a link to an Act or Work

Link to another Act or Work using [link text](frbr-uri), where frbr-uri is the FRBR URI of the work you want to link to, such as /akn/za/act/1998/2.

Repealed by [Act 5 of 2009](/za/act/2009/5).
Repealed by Act 5 of 2009.

{% endblock %} {% block cheat-remark %}
Add an editorial remark

Add an editorial remark, text that isn't part of the original document, using [[remark]].

[[An editorial remark]]
[An editorial remark]
{% endblock %} {% block cheat-escape %}
Ignore text that looks like a chapter, part or section but isn't

You can tell the system to ignore text that looks like a chapter, part, section or any other special words, by putting a backslash \ just before it. 

\1. An item that isn't a section
\(a) text that isn't a list item
\Chapter that isn't a chapter
1. An item that isn't a section (a) text that isn't a list item Chapter that isn't a chapter
{% endblock %} {% block cheat-schedule %}
Starting a Schedule

Start a schedule using SCHEDULE.

SCHEDULE
HEADING Schedule heading

This is the body of the schedule.

Include a subheading with SUBHEADING below HEADING.

SCHEDULE
HEADING First schedule 
SUBHEADING Laws to be repealed

This is the body of the first schedule.

First Schedule

Laws to be repealed

This is the body of the first schedule.
{% endblock %} {% block cheat-table %}
Insert a table

Insert a table using Wikimedia table syntax.

  • A table starts with {| and ends with |}.
  • Use |- to separate rows.
  • Use | to start a cell on a row.
  • Use ! to start a heading cell on a row.
{|
|-
! header 1
! header 2
! header 3
|-
| row 1, cell 1
| row 1, cell 2
| row 1, cell 3
|-
| row 2, cell 1
| row 2, cell 2
| row 2, cell 3
|}
header 1 header 2 header 3
row 1, cell 1 row 1, cell 2 row 1, cell 3
row 2, cell 1 row 2, cell 2 row 2, cell 3
{% endblock %} {% block cheat-convert-table %}

Convert lines of text to a table by selecting the lines, and then clicking Insert, Insert table.
